# Research Context — quantum-testing

Thu muc nay luu tru toan bo context nghien cuu de co the tiep tuc phat trien
tren bat ky may nao, bat ky luc nao.

## Cau Truc

```
research/
├── README.md                       — File nay
├── context/                        — Tong quan nghien cuu
│   ├── 00-RESEARCH-OVERVIEW.md     — Tổng quan, mục tiêu, timeline
│   ├── 01-DEEP-RESEARCH.md         — Báo cáo nghiên cứu sâu
│   ├── 02-ALGORITHM-DESIGN.md      — Thiết kế thuật toán
│   ├── 03-PAPER-OUTLINE.md         — Outline bài báo
│   ├── 04-REFERENCES.md            — Tài liệu tham khảo
│   └── 05-HARDWARE-NOTES.md        — Ghi chú phần cứng
├── design/                         — Chi tiết thiết kế (code, diagrams)
├── paper/                          — Bài báo (drafts, figures)
└── references/                     — PDF papers, notes
```

## Bat Dau Nhanh

1. Doc `context/00-RESEARCH-OVERVIEW.md` de hieu tong quan
2. Doc `context/01-DEEP-RESEARCH.md` de hieu boi canh nghien cuu
3. Doc `context/02-ALGORITHM-DESIGN.md` de hieu thiet ke thuat toan
4. Doc `context/03-PAPER-OUTLINE.md` de hieu cau truc bai bao

## Session Continuation

Khi mo lai session moi, chi can:
1. Doc `context/00-RESEARCH-OVERVIEW.md` (30 giay)
2. Doc `context/02-ALGORITHM-DESIGN.md` (1 phut)
3. Bat dau code tiep tuc

Toan bo quyet dinh thiet ke, ly do chon huong, va ke hoach deu duoc
luu tru trong thu muc nay.
